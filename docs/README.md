## 项目介绍

KfunPet（快芙桌宠）是一款基于 **WinUI** 开发的**全智能 AI 桌宠**应用。核心亮点是让桌宠拥有**自我意识**，可独立操控身体各部位；同时采用 **混合动作系统**，结合本地预设动画与 AI 实时驱动，在低成本、高性能的前提下，实现富有创意的角色交互表现。

### 核心特性

| 特性 | 说明 |
|------|------|
| 全智能 AI 桌宠 | 桌宠拥有自我意识，可独立操控身体各部位实现自主行为 |
| 混合动作系统 | 预设动画保证流畅，AI 驱动实现创意即兴动作 |
| 伪骨骼系统 | 精细化控制 Head、Body、Arm、Leg 等部件的旋转与位移 |
| 智能调度器 | 预设优先策略，优先匹配本地动作库，无命中时才调用 AI 生成，大幅节省 tokens 消耗 |
| 动作沉淀 | AI 生成的优质动作可沉淀为预设，持续扩展动作库 |
| 情绪感知 | 根据当前情绪状态自动匹配并播放对应动作 |
| 跨平台互联 | 支持与 Android 端、开发板端（如 ESP32-S3）实现桌宠互联 |

### 架构优化说明

| 新增/优化模块 | 职责 | 说明 |
|-------------|------|------|
| **感知层** | 环境感知 | 屏幕边界检测、鼠标位置追踪、用户活跃状态、时间感知，为 AI Brain 提供决策输入 |
| **AI Brain** | 自我意识 | 持续运行的意识循环，自主决定"现在要做什么"，而非仅靠事件被动触发 |
| **动作调度器** | 执行决策 | 将 AI Brain 的意图转化为具体动作序列，负责预设优先查找与 AI 动作生成 |
| **记忆/学习模块** | 知识沉淀 | 存储用户偏好、动作历史、经验学习，实现"持续进化" |
| **通信模块** | 跨平台互联 | 支持蓝牙、WiFi、互联网通信，实现 PC/Android/开发板三端互联 |
| **情绪系统** | 状态管理 | 根据当前情绪状态选择合适的动作表现，增强角色感染力 |

### 动作调度策略

系统采用 **预设优先** 的调度策略，以最小化 AI 调用成本：

```
动作请求 → 查询预设动作库 → 命中? → 是 → 直接播放预设动画
                              ↓ 否
                    调用 AI 生成动作序列
                              ↓
                    播放 AI 生成的动画
                              ↓
                    (可选) 沉淀为预设
```

- **基础动作**（打招呼、眨眼、伸懒腰等）直接调用预设动作库，零 AI 成本
- **AI 提议动作** 先查预设库，命中则复用，未命中才让 AI 自主生成
- **动作沉淀**：AI 生成的优质动作可保存到预设库，避免重复消耗 tokens

## 未来构想

### 构想一：跨平台桌宠互联生态

打造 **PC + Android + 开发板** 三端互联的桌宠社交网络。

- **设备互联**：当用户拥有多台安装桌宠的设备时，可选择是否开启互联。开启后，不同设备间的桌宠可以互相"串门"，在设备间自由穿梭。
- **用户互联**：用户之间可以互相添加好友，好友的桌宠之间也能实现互动与拜访。
- **陌生人随机访问**：用户可选择开启"随机串门"功能，系统会随机匹配其他在线用户，让你有机会遇到陌生人的桌宠，带来意外的惊喜与互动。
- **跨平台互联**：PC 桌宠可与 Android 端、开发板端的桌宠实现无缝互联，共享互动体验。

> **注**：Android 端和开发板端为独立项目。

### 构想二：开发板端二次元桌面时钟

基于 **ESP32-S3 等开发板** 开发独立运行的二次元桌面时钟硬件。

- **基础功能**：保留传统桌面时钟的时间显示，支持多种时钟界面风格。
- **融入桌宠**：在时钟界面中嵌入桌宠模块，让时钟不再单调，桌宠可陪伴你的每一次查看。
- **设备互联**：开发板时钟可与 PC 桌宠、Android 桌宠及其他开发板设备实现互联，共享桌宠互动体验。
- **通讯方式**：
  - **蓝牙 / WiFi**：近距离设备间直连，低延迟、高私密。
  - **互联网**：远距离跨网段通信，随时随地远程互联。

### 互联体系总览

| 设备 | PC 桌宠 | Android 桌宠 | 开发板端（如 ESP32-S3） |
|------|--------|-------------|-----------------------|
| PC 桌宠 | ✅ 设备间互访 | ✅ 跨平台互联 | ✅ 跨平台互联 |
| Android 桌宠 | ✅ 跨平台互联 | ✅ 设备间互访 | ✅ 跨平台互联 |
| 开发板端 | ✅ 跨平台互联 | ✅ 跨平台互联 | ✅ 设备间互访 |
| 陌生人随机访问 | ✅ | ✅ | ✅ |
| 近距离（蓝牙/WiFi） | — | ✅ | ✅ |
| 远距离（互联网） | ✅ | ✅ | ✅ |

## 愿景

- **全智能自我意识**：赋予桌宠真正的自我意识，可独立操控身体各部位，实现自主思考与行为。
- **对话交互**：用户可通过自然语言与 AI 进行对话，获得更丰富的互动体验。

- **跨平台互联**：打通 PC、Android、开发板三端，让桌宠在不同设备间自由穿梭。
- **数字生命**：让 AI 不再是冰冷的代码，而是一个更具“生命力”的数字“人”
