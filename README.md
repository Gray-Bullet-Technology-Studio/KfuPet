## 说明

> [!IMPORTANT]
>
> ### 关于本项目
>
> - 本项目 **KfuPet** 是一款基于 **WinUI** 开发的桌面宠物应用
> - 采用 **混合动作系统**，结合本地预设动画与 AI 实时驱动，实现低成本、高性能且富有创意的角色交互表现
> - 本项目采用 **AGPL-3.0** 协议 + 自定协议进行开源

## 🧑‍💻 开发

- 本项目采用 **C#** + **WinUI** 开发（基于 "WinUI 空白应用（已打包）" 模板）
- Windows SDK 版本要求：>= 10.0.22621.0

### 快速开始

1. 使用 Visual Studio 打开项目
2. 还原 NuGet 依赖
3. 启动调试：`F5`
4. 构建：右键项目 → 生成
